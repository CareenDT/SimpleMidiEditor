SIMPLEMIDIEDITOR (SMIDIE)

SMIDIE - простой но функциональный редактор и проигрыватель .MDI файлов.

Программа создана для минималистичного создания музики и изучения принципов DAW программ.

В приложение будут встроены функции:
  - Создание и проигрывание MIDI последовательности
  - Выбор инструментов
  - Высокая кастомизация редактора
  - Понятный интерфейс
  - Удобное перемещение по последовательности
  - Изменение размеров сетки
  - Широкое изменение параметров проекта

Приложение имеет свой тип файла под проекты.
MIDI проект можно конвертировать в MDI файл

Реализовано:
  - Создание и проигрывание MIDI последовательности
  - Удобное перемещение по последовательности
  - Изменение размеров сетки
  - Простая кастомизация редактора в виде его оформления
  - Понятный интерфейс (~)
  - Параметры проекта(10%)
  - Расстановка нот, без удаления
