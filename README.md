SimpleMIDI EDITOR (SMIDIE)

Проект редактирования MDI файлов - Создание, Изменение.

**Цели и задача проекта:**

  - Использование существующих MDI файлов и их последующее редактирование
  - Создание MDI файлов для заполнения
  - Приложение ориентировано на лёгкую обработку и простоту управления.
  - Проигрывание и взаимодействие с MIDI в реальном времени
  - Выбор всех базовых интрументов MIDI
  - Настраиваемый и разборчивый интерфейс
  - Кросс-платформенность
  - Глубокая настройка MIDI, файлов проекта, а также самого редактора.
  - Сохранение проектов со всеми настройками
  - Автосохранение файлов проекта
